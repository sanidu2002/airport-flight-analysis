# Airport Flight Analysis System

A first-year Python coursework project developed for the University of Westminster.

## Features

- Load airport departure data from CSV files
- Validate airport and year inputs
- Calculate flight statistics
- Generate operational reports
- Save results to text files
- Display airline departure histograms using Python graphics

## Technologies

- Python
- CSV Module
- John Zelle Graphics Library

## Sample Data

- CDG2021.csv
- LHR2025.csv
